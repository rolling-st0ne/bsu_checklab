Чеклаб by ИИ-шки

Repack byXatab:

  • Программы сдаются в виде ZIP архива с названием {номер лабы}_{фамилия латиницей}.zip, например, 1_atrokhau.zip
  • Внутри папка с тем же названием
  • В папке должны быть только .py файлы из лабораторной, не должно быть подпапок или скрытых элементов
  • Размер архива не должен превышать 32 КБ
  • Программы должны использовать только разрешенные модули
  • Использование input() под запретом – все параметры в программах должны передаваться в командной строке (и считываться из массива sys.argv)
  • Формат входных и выходных параметров также должен совпадать с указанным в задании
